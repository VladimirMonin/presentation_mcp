# Архив старых файлов

Эта папка содержит оригинальные монолитные скрипты, которые были рефакторены в новую модульную систему.

## Старые файлы

- **one.py** — анализатор шаблонов PPTX
- **two.py** — очиститель Markdown текста
- **three.py** — генератор презентаций

## Что изменилось

Старые скрипты были полностью переработаны в модульную систему:

### Было (3 монолитных файла)
```
one.py    → 150+ строк анализа шаблонов
two.py    → 50+ строк очистки Markdown
three.py  → 300+ строк генерации презентаций
```

### Стало (5 пакетов)
```
config/        → настройки и регистрация макетов
core/          → бизнес-логика (builder, cleaner, processor, analyzer)
models/        → типобезопасные модели данных
io_handlers/   → работа с файлами и путями
cli/           → командный интерфейс
tests/         → unit тесты с pytest
```

## Миграция

См. документацию по миграции: [`../doc/MIGRATION.md`](../doc/MIGRATION.md)

### Запуск старых скриптов (не рекомендуется)

```bash
# Анализ шаблона
python archive/one.py template.pptx

# Генерация презентации
python archive/three.py config.json
```

### Запуск новой системы (рекомендуется)

```bash
# Анализ шаблона
python main.py analyze --template template.pptx

# Генерация презентации
python main.py generate --config config.json
```

## Преимущества новой системы

1. **Модульность**: четкое разделение ответственности
2. **Тестируемость**: полное покрытие unit тестами
3. **Расширяемость**: легко добавлять новые функции
4. **CLI**: удобный интерфейс командной строки
5. **Markdown**: поддержка MD файлов для заметок
6. **Пути**: гибкая работа с абсолютными/относительными путями
7. **Документация**: подробные руководства и примеры
