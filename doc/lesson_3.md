# Описание ключей

- `project`: Название проекта или курса.
- `lesson`: Номер урока.
- `lesson_dir`: Путь к файлу с материалами урока.
- `title`: Заголовок урока.
- `objective`: Цель урока.
- `outcome`: Ожидаемый результат после прохождения урока.
- `research_needed`: Флаг, указывающий, требуется ли исследование для урока
- `research_topics`: Список тем для исследования.
- `research_links`: Список файлов с готовыми исследованиями (наполняется после проведения исследования).
- `code_samples`: Флаг, указывающий, есть ли примеры кода для урока. (Может быть строкой или списком путей к файлам с примерами кода).
- `long_read`: Флаг, указывающий, готов ли лонгрид для урока.
- `presentation_dir`: Путь к директории с презентацией для урока.
