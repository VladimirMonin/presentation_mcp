# Макет: `shorts_fullscreen`

## Описание

**Вертикальное изображение на весь слайд (9:16)** — макет для создания контента для вертикальных видеоплатформ.

## Параметры

- **Тип:** Полноэкранный вертикальный
- **Количество изображений:** 1
- **Соотношение сторон:** 9:16 (вертикальное)
- **Размер слайда:** 19.05 см × 33.867 см

## Координаты размещения

| Параметр | Значение |
|----------|----------|
| **Left** | 0.0 см |
| **Top** | 0.0 см |
| **Max Width** | 19.05 см |
| **Max Height** | 33.867 см |

**Покрытие:** 100% площади слайда (без полей)

## Рекомендуемые размеры изображений

### Идеальное разрешение
- **1080×1920 пикселей** (Full HD вертикальное)
- **2160×3840 пикселей** (4K вертикальное)

### Соотношение сторон
- **9:16** — идеально (без обрезки)
- **3:4** — допустимо (будут черные полосы сверху/снизу)
- **1:1** — допустимо (будут черные полосы сверху/снизу)

⚠️ **Важно:** Изображения с другим соотношением сторон будут масштабированы с сохранением пропорций, что может привести к появлению пустого пространства.

## Целевое использование

Этот макет предназначен для создания контента для:

- ✅ **YouTube Shorts** (вертикальные видео до 60 секунд)
- ✅ **TikTok** (основная платформа вертикального контента)
- ✅ **Instagram Reels** (короткие вертикальные видео)
- ✅ **Instagram Stories** (временный контент 9:16)
- ✅ **Facebook Stories**
- ✅ **Snapchat**

```json
{
  "template_path": "templates/youtube_base_shorts.pptx",
  "layout_name": "ShortsLayout",
  "output_path": "shorts_presentation.pptx",
  "slides": [
    {

```

## Особенности работы

### Текстовые заполнители

⚠️ **Важно:** Шаблон `youtube_base_shorts.pptx` **не содержит** текстовых заполнителей (заголовок, номер слайда).

Слайды полностью состоят из:
- Изображения (100% покрытие)
- Аудио (опционально)

Поле `title` в JSON используется только для:
- Логирования (отладка)
- Внутреннего именования слайдов

**Текст на слайде не отображается!** Весь контент должен быть запечен в изображение.

### Аудио

Поддерживается автоматическое воспроизведение аудио:
- Форматы: **MP3, WAV, M4A, AAC**
- Воспроизведение: автоматически при открытии слайда
- Циклирование: нет (одноразовое воспроизведение)

Подробнее см. [`doc/technical/media_autoplay.md`](../technical/media_autoplay.md)

## Workflow для создания Shorts

### Шаг 1: Подготовка материалов

1. **Изображения:**
   - Создайте вертикальные изображения 1080×1920 пикселей
   - Весь текст/графику запеките в изображение (используйте Figma, Canva, Photoshop)

2. **Аудио (опционально):**
   - Подготовьте озвучку или фоновую музыку
   - Форматы: MP3, WAV

### Шаг 2: Создание конфигурации

Создайте JSON файл (например, `shorts_config.json`):

```json
{
  "template_path": "templates/youtube_base_shorts.pptx",
  "layout_name": "ShortsLayout",
  "output_path": "my_shorts.pptx",
  "slides": [
    {
      "slide_type": "content",
      "layout_type": "shorts_fullscreen",
      "title": "Slide 1",
      "notes_source": "",
      "images": ["slide1.png"],
      "audio": "audio1.mp3"
    }
  ]
}
```

### Шаг 3: Генерация презентации

```bash
python main.py generate shorts_config.json
```

### Шаг 4: Экспорт в видео

PowerPoint позволяет экспортировать презентацию в видео:

1. **Файл** → **Экспорт** → **Создать видео**
2. Выберите качество: **Full HD (1080p)**
3. Время на слайд: установите по длительности аудио
4. **Создать видео** → сохраните как MP4

**Результат:** Готовое вертикальное видео 9:16 для загрузки на YouTube Shorts, TikTok и т.д.

## Устранение проблем

### Изображение не заполняет весь слайд

**Причина:** Соотношение сторон изображения не 9:16.

**Решение:**
1. Проверьте размер изображения (должно быть 1080×1920)
2. Обрежьте или измените размер в графическом редакторе
3. Используйте онлайн-инструменты (например, iloveimg.com/resize-image)

### Текст не отображается на слайде

**Это нормально!** Шаблон не содержит текстовых заполнителей.

**Решение:** Запеките весь текст в изображение перед генерацией.

### Аудио не воспроизводится автоматически

**Возможные причины:**
1. Неподдерживаемый формат (используйте MP3)
2. Файл не найден (проверьте путь)
3. Открыто в PowerPoint Online (автовоспроизведение не работает в веб-версии)

**Решение:** Проверьте логи (`logs/app.log`) и убедитесь, что аудио было успешно добавлено.
